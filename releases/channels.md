# Release Channels

| Channel | Audience | Trust and qualification |
| --- | --- | --- |
| Internal Preview | Developers and selected evaluators | Narrow boundary; may be unsigned; no production use |
| Closed Beta | Selected design partners | Beta blockers closed; monitored installed-artifact soak |
| Public Beta | Public pre-production users | Signed distribution, security review, support policy |
| Stable | Production users | Full supported matrix, operations, recovery, approval |
| LTS | Long-lived production users | Stable plus published maintenance and end-of-support dates |

Channel promotion is one-way only after evidence gates pass. A feature can
remain experimental even when the surrounding release advances.
