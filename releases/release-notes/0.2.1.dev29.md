# PersistMind 0.2.1.dev29 - Internal Windows Preview

Released July 19, 2026 for restricted local testing.

## Summary

This candidate moves PersistMind to its next-generation application and
eight-authority storage architecture. It also incorporates migration,
concurrency, Windows path, PowerShell, branding, uninstall, and release
hardening completed since the previous published preview.

## Architecture and reliability changes

- Separates control, source, task, activity, audit, knowledge, policy, and
  learning authority stores behind application ports.
- Publishes immutable source segments in the content-addressed object tree.
- Tightens migration verification, concurrency fencing, path handling, and
  PowerShell installer behavior.
- Keeps stdio MCP read-only and exposes 92 explicitly registered read tools.
- Fixes staged restore when source-segment images and the copied object tree
  share `objects/sha256`.

## Qualification

The exact Drive wheel passed a fresh Windows 11 Pro/Python 3.13 installation,
project setup, indexing, doctor, full storage integrity, source search, context
pack, Codex read preflight, read-only MCP registration, encrypted backup,
backup verification, staged restore, and safe uninstall checks. The root
PowerShell installer also passed its packaged `-VerifyOnly` check.

The test host's Windows Credential Manager was saturated by pre-existing test
credentials. Backup and restore were therefore exercised from the exact
installed wheel with PersistMind's explicit development key-provider switch;
the production keyring allocation path was not observed in this run.

## Restrictions

Use only disposable or non-critical local repositories. Manual review is
mandatory. Windows 10, Python 3.11/3.12, Linux, and macOS remain unqualified for
this candidate. The historical source suite still includes tests for retired
monolithic APIs, so a clean full-suite claim is not made.

Remote writes, writable MCP, autonomous source modification, automatic
self-improvement adoption, team/server mode, cross-repository authority, and
production-authoritative experimental intelligence remain unsupported. The
trusted updater must not install this unsigned preview.
