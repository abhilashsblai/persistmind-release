## Qualification

The exact same wheel passed clean installed-package smoke on Windows 11 with
Python 3.11, 3.12, and 3.13. Source tests, identity/profile enforcement,
indexing, file changes, search, packs, workflow/scope, memory, storage

## Restrictions

Use only disposable or non-critical local repositories. Manual review is
mandatory. Remote writes, writable MCP, autonomous source modification,
automatic self-improvement adoption, team/server mode, cross-repository
authority, and production-authoritative advanced intelligence are unsupported.

The official signing key was not requested, copied, exposed, or bypassed. The
trusted updater was not weakened and must not install this unsigned preview.
