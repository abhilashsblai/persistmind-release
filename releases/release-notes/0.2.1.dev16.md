Released July 17, 2026 for restricted local testing.

real FoxFlow workflow. It supersedes dev15 without overwriting that immutable

## Fixed

- Legacy non-UTF-8 bytes in a committed Git diff no longer crash repository
  identity and source indexing.
- Valid UTF-8 dirty-worktree identity remains compatible with earlier builds.
- Codex preflight is routed through split-v1 when split Task State is active.
- Plan pack IDs must be finalized in split Activity and match the task binding.
- CLI and MCP `seed_from_diff` requests reach split pack assembly.
- `verification_incomplete` survives concise CLI rendering instead of appearing
  as an internal product failure.

## Qualification

The exact wheel passed installation and lifecycle qualification on Windows 11
with Python 3.12 in a disposable Git repository. The smoke covered setup,
storage, indexing, search, packs, task/plan/checkpoint flow, scoped verification,
outcome/continuation, memory provenance, read-only MCP, backup/restore, restart,
and uninstall preservation.

The source-focused release checks also passed compilation, Ruff, generated
interface synchronization, and focused preview/runtime tests. A broad regression
campaign was intentionally not used for this candidate; real-project feedback is
the next acceptance signal.

## Restrictions

Use only disposable or non-critical local repositories. Manual review is
mandatory. Remote writes, writable MCP, autonomous source modification,
automatic self-improvement adoption, team/server mode, and production use remain
unsupported. Windows 10, Python 3.11/3.13, Linux, macOS, and signing qualification
are pending.
