# PersistMind 0.2.1.dev19 - Internal Windows Preview

Released July 17, 2026 for restricted local testing.

## Summary

This release corrects the installable ZIP contract after following the published
GitHub release instructions against FoxFlow exposed a concrete dev18 packaging
failure. The ZIP stored the installer under `documentation/` and the wheel under
`artifacts/`; the documented root command could not find the installer, while the
nested installer could not find its wheel.

## Fixed

- `Install-PersistMind.ps1` and `INSTALL-WINDOWS.txt` are present at the
  extracted ZIP root.
- The installer resolves the exact wheel beside a standalone installer or under
  the extracted bundle's `artifacts` directory.
- The packager runs the documented root installer with `-VerifyOnly` before ZIP
  creation and fails publication if that command is not usable.

## Verification

The dev18 failure was reproduced through both the documented root command and
the nested installer. The dev19 ZIP then passed checksum verification, root
installer verification, a fresh install into `C:\PersistMind\0.2.1.dev19`,
FoxFlow setup and MCP probes, `doctor`, the existing governed preflight, and the
formerly blocked Supabase write-channel hook.

## Restrictions

and production-updater qualification remain pending.
