# Release Channels

| Channel | Audience | Status | Artifact location |
| --- | --- | --- | --- |
| Internal Preview | Approved testers | Current | Private Google Drive channel |
| Closed Beta | Selected design partners | Planned | Controlled Google Drive channel |
| Public Beta | Public evaluators | Not available | To be announced |
| Stable | Production users | Not available | To be announced |
| LTS | Long-lived production users | Not available | To be announced |

The current Internal Preview has a narrow Core Local boundary, may be unsigned,
and is not approved for production use. Later channels require the qualification
gates defined in the [qualification policy](qualification-policy.md).

Channel promotion is one-way only after evidence gates pass. A feature can
remain experimental even when the surrounding release advances.
