# Support

## Current support boundary

PersistMind `0.2.1.dev31` is an internal Windows preview for disposable or
non-critical local repositories. It requires manual review and has no
production, enterprise, response-time, or compatibility SLA.

## Getting help

1. Read [getting started](docs/getting-started.md),
   [troubleshooting](docs/troubleshooting.md), and the
   [current known issues](docs/known-issues.md).
2. Search existing issues in the
   [release repository](https://github.com/abhilashsblai/persistmind-release/issues).
3. Open an issue containing the version, source commit, artifact SHA-256,
   Windows and Python versions, command, expected result, actual result, and a
   redacted reproduction.

Use [SECURITY.md](SECURITY.md), not a public support issue, for vulnerabilities.
Commercial or production use requires a separate agreement under the license.
