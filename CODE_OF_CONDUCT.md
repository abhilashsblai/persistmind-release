# Code of Conduct

## Our standard

Participate with respect, patience, technical honesty, and consideration for
different experience levels. Discuss evidence and tradeoffs without personal
attacks. Do not harass, threaten, discriminate, expose private information, or
publish another person's sensitive repository data.

## Scope and enforcement

This standard applies to repository issues, pull requests, discussions, and
other project spaces. Report conduct concerns privately to the repository owner
through their GitHub profile. Maintainers may edit or remove content, close
threads, or restrict participation when necessary to protect the community.
