# Current Limitations

PersistMind `0.2.1.dev17` has these intentional boundaries:

- private internal testing only;
- unsigned artifacts and manual checksum verification;
- Windows 11 directly observed; Windows 10 not directly observed;
- Python 3.12 directly qualified; Python 3.11 and 3.13 pending;
- Linux and macOS qualification pending;
- disposable or non-critical local repositories only;
- manual human review for all agent-assisted work;
- local filesystem and local Git repositories;
- CLI mutations only, with read-only stdio MCP;
- no remote team service, remote authorization, or mutation broker;
- no writable MCP or caller-controlled authority escalation;
- no autonomous source modification;
- no automatic self-improvement adoption;
- no production-authoritative advanced intelligence; and
- no automatic updater installation for this unsigned artifact.

See [known issues](known-issues.md) for observed gaps and
[roadmap](roadmap.md) for planned qualification phases.
