# CTX Layer 0.2.0a13

Released July 12, 2026.

This is the updater-adoption preview release. Machines running an older CTX
Layer wheel install `0.2.0a13` once. Future project updates use:

```text
ctxlayer update
```

## Highlights

- Signed release manifests and exact SHA-256 wheel verification.
- Immutable engine slots with per-project version bindings.
- Candidate installation and health checks before project mutation.
- Verified backup, crash journal, atomic cutover, and rollback.
- Legacy `workspace.db` migration into seven split-storage roles.
- Durable knowledge, learning, task, activity, audit, and policy preservation.
- Source chunks, vectors, snapshots, and indexes rebuilt after durable migration.
- Original monolith and verified migration backup retained after success.

## Install Or Adopt The Updater

```powershell
python -m pip install --upgrade "https://github.com/abhilashsblai/ctxlayer-release/releases/download/v0.2.0a13/ctxlayer-0.2.0a13-py3-none-any.whl"
```

## Update A Project

```text
cd <project>
ctxlayer update --check
ctxlayer update
```

Use `ctxlayer update --dry-run`, `ctxlayer update status`, or
`ctxlayer update --rollback` for validation, status, and protected rollback.
