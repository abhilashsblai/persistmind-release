# Integrations

PersistMind works alongside coding agents and editors; it does not replace them.

## Current supported boundary

- local CLI workflows;
- local read-only stdio MCP;
- absolute repository binding; and
- manual human review of agent actions and diffs.

## Agent guides

- [Codex](../guides/codex.md)
- [Claude Code](../guides/claude.md)
- [Cursor](../guides/cursor.md)
- [VS Code](../guides/vscode.md)

The current release does not claim remote integrations, writable MCP, hosted
connectors, lifecycle automation, or autonomous mutation authority. Integration
availability must be stated per release and client.
