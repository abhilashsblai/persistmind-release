# Glossary

**Artifact** - A built release file such as a wheel, source distribution, or ZIP.

**Candidate** - One immutable version and source commit undergoing qualification.

**Context pack** - A bounded, deterministic collection of repository context,
provenance, and workflow information for a task.

**Core Local** - The local repository, local filesystem, CLI, human-reviewed
operating boundary.

**Evidence bundle** - Machine-readable and human-readable proof attached to a
release qualification.

**Internal Preview** - A narrow, non-production evaluation channel with explicit
limitations.

**MCP** - Model Context Protocol. The current preview exposes only read-only
stdio operations.

**Runtime profile** - A fail-closed configuration that defines enabled authority
and maturity boundaries.

**Snapshot** - An identified repository/index state used for provenance and
deterministic context.

**Trusted updater** - An update path that authenticates signed release metadata
and verifies exact artifact identity before installation.
