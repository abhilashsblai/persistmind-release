# Changelog

PersistMind uses detailed release notes rather than an unqualified feature
ledger. See [releases/release-history.md](releases/release-history.md).

## 0.2.1.dev4 - 2026-07-17

- Introduced the locked `windows-internal-preview` runtime profile.
- Added installed-wheel qualification on Windows with Python 3.11, 3.12, and
  3.13.
- Published explicit release identity, checksums, manifest, limitations, and
  qualification evidence.
- Restricted distribution to version-specific Google Drive artifacts.
- Kept remote writes, writable MCP, autonomous source modification, and
  automatic self-improvement adoption outside the supported boundary.
