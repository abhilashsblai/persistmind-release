# Contributing

This repository is the documentation and release-record surface for
PersistMind. Product source changes belong in the
[PersistMind source repository](https://github.com/abhilashsblai/PersistMind).

Documentation contributions should:

- describe only behavior supported by evidence;
- distinguish implemented, experimental, disabled, and release-qualified
  capabilities;
- keep current-release links version-specific;
- avoid credentials, sensitive source, private paths, and signing material;
- never add wheels, installers, archives, or other release bytes to GitHub;
- update navigation when adding or moving a page; and
- use clear, platform-specific commands with safety restrictions.

Before proposing a change, check Markdown links, confirm artifact hashes against
the release manifest, and search the tracked tree for stale product/version
names. Follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
